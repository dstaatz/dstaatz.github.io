Today is the day to learn how to write "Hello World" in Rust.
<!-- more -->

Here you go:

```rust
fn main() {
    println!("Hello World");
}
```

Now we can just run this in release mode (gotta go fast):
```
$ cargo run --release
Hello World
$
```
